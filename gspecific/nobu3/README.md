# nobu3 Python 스크립트

- `gen_font_from_db_nobu3.py`: NB3 인명, 지역명, 계절 단어에서 필요한 글자 조합을 수집해 `nobu3-BISCO.bmp` 폰트와 `code.json`을 생성한다.
- `assign_code_from_code_db.py`: 인명·지역·기타 DB와 `code.json`을 기준으로 스크립트의 문자열을 커스텀 폰트 토큰으로 치환한다. 문자열 공간이 부족할 때는 원본 바이너리 뒤쪽의 널 영역을 확인한 후 주소 범위를 확장한다.
- `expand_name_based_on_dict.py`: 이름·지역 사전의 한국어 표기를 2글자 폰트 토큰으로 변환하고, 일본어 원문과의 길이 차이에 맞춰 스크립트 주소 및 널 패딩을 조정한다. 현재 `ws_num = 4`이므로 NB4 작업에도 재사용되는 형태이다.
- `expand_name_based_on_family_name.py`: NB3의 성과 지역명을 변환하면서 이름과 분리하기 위한 1바이트 공백을 보장한다. 대상 파일별 주소 범위를 제한하고 원본 바이너리의 빈 공간을 확인한 뒤 JSON을 갱신한다.
- `name_check.py`: 스크립트에서 연속된 성/이름 항목을 읽어 `NameDB` 등록 여부와 한국어 표기의 불일치를 점검한다. 현재 설정은 NB5 진단용이며 저장 코드는 주석 처리되어 있다.

모든 스크립트의 작업 공간 및 DB 경로가 코드에 고정되어 있으므로 `ws_num`, `game`, 대상 파일과 경로를 확인한 뒤 실행해야 한다.
